Place your trained "Hey Hermes" wake word model here.

1. Sign up at https://picovoice.ai/ for a free access key
2. Train "Hey Hermes" at https://console.picovoice.ai/
3. Download the Android model file and save it as:
     hey-hermes_en_android_v3_0_0.ppn

4. Set your access key in ~/.gradle/gradle.properties:
     picovoiceAccessKey=YOUR_KEY_HERE

Without these, the app falls back to tap-to-talk mode (no wake word).
